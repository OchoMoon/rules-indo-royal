INDO ROYAL ROLEPLAY

Peraturan Server

  • Dilarang menggunakan cheat, mod ilegal, atau tools pihak ketiga.
  • Wajib menjaga roleplay sesuai karakter masing-masing.
  • DM (Deathmatch) tanpa alasan roleplay sangat dilarang.
  • Dilarang spam, toxic, atau menghina pemain lain di OOC/IC.
  • Gunakan nama karakter sesuai format RP (Contoh: Andi_Saputra).
  • Bug abuse, exploit atau glitching akan dikenakan sanksi berat.
  • Hormati admin dan semua pemain, jaga komunitas tetap sehat.
